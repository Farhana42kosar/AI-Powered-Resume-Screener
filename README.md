# AI Powered Resume Screener

An intelligent resume screening system built using Machine Learning and Streamlit.  
The application analyzes resume content and predicts the most relevant job domain/category.

##  Live Demo
https://ai-powered-resume-screener-bf7kjkkb33pk7jsetkwofj.streamlit.app/


##  Features
- Upload resumes in PDF/DOCX format
- Extracts and cleans resume text
- Uses ML model for domain prediction
- Instant results
- Simple and interactive UI
- Deployable on cloud



##  Tech Stack
- Python
- Streamlit
- Scikit-learn
- TF-IDF Vectorizer
- ML
- Pandas / NumPy


##  How It Works
1. User uploads resume.
2. Text is extracted from the file.
3. Preprocessing & vectorization are applied.
4. Trained ML model predicts the job category.
5. Result is displayed on screen.
